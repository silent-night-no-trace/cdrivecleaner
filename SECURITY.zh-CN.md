# 安全说明

如果你认为在 `CDriveCleaner` 中发现了安全问题，请先不要公开披露，建议在维护者有机会评估前以非公开方式说明。

## 报告渠道

推荐方式：使用本仓库 GitHub 的私密漏洞报告入口。

- 打开仓库的 `Security` 标签页
- 点击 `Report a vulnerability`

如果当前仓库尚未启用私密报告入口，请先开一个 GitHub issue 仅说明“存在潜在安全问题，需要私下沟通”（不要包含可利用细节），并请求维护者提供私密沟通渠道。

## 建议提供的信息

- 受影响版本或提交
- 复现步骤
- 影响范围说明
- 如已知，可附上缓解建议

## 当前说明

仓库仍可能在迭代完善披露与分级响应流程。在流程未完全固定前，优先使用 GitHub 私密报告入口（如可用）。
