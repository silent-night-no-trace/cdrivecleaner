# Security Policy

If you believe you have found a security issue in `CDriveCleaner`, please avoid public disclosure before maintainers have time to assess the report.

## Where to report

Preferred: use GitHub's private vulnerability reporting for this repository.

- Go to the repository's `Security` tab.
- Click `Report a vulnerability`.

If private reporting is not enabled yet, open a GitHub issue that only says you have a potential security report (do not include exploit details) and request a private channel.

## Recommended report contents

- affected version or commit
- reproduction steps
- impact summary
- suggested mitigation if known

## Current note

This repository may still be iterating on disclosure and triage workflow details. Until those are finalized, prefer GitHub private reporting when available.
